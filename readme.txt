== Description ==
Manage back-end admin panel design style by a single css file and overwrite existing css.
For more information about the plugin please see: http://www.krishe.com/en/admin-css

== Installation ==

1. Upload `admin-css` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= What does this plugin do, exactly? =
Edit admin.css file and change wordpress default back-end theme.
